About the Game

Five Nights at Derpos (FNaD) is a fan-made horror game inspired by the Five Nights at Freddy’s series...

In FNaD, you take on the role of the unlucky night guard at various Derpo's locations...

Each installment in the FNaD series features a twist on FNaF’s original mechanics and gameplay...

About the Programming

FNaD was built entirely using Scratch, a visual programming platform...

The game’s core systems (camera switching, power management, animatronic AI, etc.) were all crafted with Scratch...

Here’s a quick breakdown of how my workflow usually goes:

  1. Design the sprites – Things usually start as super basic placeholder images...
  2. Program the interactions – Kind of hard to play a game that’s just images...
  3. Add sounds – I’m not a sound designer, so I’ll be real...
  4. Test and refine gameplay – After way too much work, it’s time to make sure the game is actually fun...
